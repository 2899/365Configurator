# 🍰 Office365 download/install tool

## 功能简介：

```
1.下载Office365ProPlus离线安装包。
2.离线安装Office365ProPlus，支持自定义组件。
3.在线安装Office365ProPlus，支持自定义组件。
4.增加程序语言-英语，在非简体中文系统上自动切换语言为英语。
5.增加更多可下载的Office365语言。
```

## 使用说明：

```
 ****请不要把批处理放在磁盘根目录！！！****
1.我没有365安装包，需要下载
  可以通过本程序直接下载(如果下载失败请转百度云：http://pan.baidu.com/s/1bo0LESB 密码：natl 离线包已内置本程序)
2.我有365安装包，想用本程序部署
  将完整的365安装包(名为Office的文件夹)放到本目录即可，运行程序后，会自动识别安装信息。
  一个完整的安装包应该包括且仅包括这些文件：
   Office
      │  
      └─Data
          │  v32.cab                        注：v32和x86是安装包的位数
          │  v32_16.0.7167.2047.cab             16.0.7167.2047是程序版本
          │                                     en-us是程序语言
          └─16.0.7167.2047                      请根据自己的安装包区分！！！
                  一些　*．CAB文件
                  stream.x86.en-us.dat
                  stream.x86.x-none.dat
3.下载失败怎么办？
   可能的原因：网络不稳定；RAM不足（需要约1GB空闲）
   解决办法：换网；清理RAM；转百度云下载离线包
```
